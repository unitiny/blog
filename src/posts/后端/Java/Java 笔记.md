## 知识点



### 运行

```
javac 会编译成类，java 运行类，如果类有包名，如parent.children.eat,则需要cd回到parent的上级目录，
然后java parent.children.eat才能运行该eat类
```



### 方法类型

```
静态方法不需要实例化对象，即可通过类名调用
非静态方法需要实例化的对象才能使用
```
