## 注意点：

### 更新data文件出错

尽量少通过wps导出文件，容易格式不对，文本类直接复制粘贴即可

### .sh文件是什么

```
其实就是在linux上的可执行*文件*，如windows上的exe*文件*一样
```

### web和www有什么区别，为什么要www

```
web动态多人访问会给服务器造成压力，所以静态www用来分担web的访问量
```
